HTML

  • The head element contains 'meta data', infomation that the user doesn't see but helps browsers find and use the page.
  • The body element contains all the information that will be displayed on the page.
  • One element nested inside another element is called a child element.
  • Inside img tags, alt attributes are always used to assist visually impared users and search engines.
  • HTML stands for HyperText Markup Language.
  • The link tag is used to attach an external document to you HTML file.
  • The script tag is placed at the bottom of the page before the closing body tag.
    • This is so the page can fully load before the JavaScript tries to access elements.
  • Tags can be semantic ie 'section', 'header', 'footer'. This provides more information to the browser about the content of these tags.
  • Other tags like 'div' and 'br' have no semantic meaning but can still be used to seperate a page into different elements.

CSS

  • CSS stands for Cascading Style Sheets.
  • It is used to make the page visually appealing and easier to read.
  • CSS can also be used to pages display properly on multiple screen sizes, like Desktops and Mobile Phones.
  • CSS can be written inside the HTML document using style tags, but it is better practice to link to an externall CSS file.
  • Using CSS makes it easier to change the look of multiple elements with minimal changes to the code.
  • Margin = Area outside of an element. Padding = Space around the content inside the element.
  • A border is drawn around the outside of the padding but the margin is outside of the border.
  • CSS takes identifiers like 'h1' or 'section' and allows to to assign properties and attributes to style them.
  • You can create your own identifiers by using class and/or id in the HTML and assigning it a name.
    • A class can be used for multiple elements and is accessed in CSS by .className
    • An ID should only be used for one element and is accessed in CSS by #idName
  • If an element has both a Class and an Id, the CSS will prioritize the id rules as they are more specific to the element.

GIT

  • GIT is a version Control System.
  • It allows multiple team members to make changes to code and allows changes to be checked before being merged.
  • GIT is often used via a CLI (Command Line Interface). Some common commands are:
    • git checkout -b branch-name : Creates a new branch and switches to it.
    • git status : Checks what branch we are currently on and shows modified files.
    • git clone [url] : Copies a repository from the URL and creates a local copy on your maching.
    • git commit -m "Description" : Commits your changes .
  • Git Pages allows you to make you site viewable on a browser, but it must contain an index.html file at the root.

JavaScript

  • A variable can store data in the for of Strings, Characters, Ints, Floats, Arrays and Booleans.
  • An Array can store multiple chunks of data.
  • A for loop will repeat for a predetermined number of times.
  • An if or if-else loop will exectute if the condition returns True.
